# Security Policy

## Supported versions

| Line | Status |
|---|---|
| `Configurite 10.x` (`net10.0`) | ✅ Active |
| `Configurite 9.x` (`net9.0`) | ✅ Active |
| `Configurite 8.x` (`net8.0`) | ✅ Active (LTS) |
| Earlier | ❌ Not supported |

## Reporting a vulnerability

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Instead, use **GitHub Security Advisories** on this repository:

1. Go to the **Security** tab.
2. Click **Report a vulnerability**.
3. Fill out the form. We'll be notified privately.

You can expect:

- Acknowledgement within **72 hours**.
- A coordinated-disclosure window of up to **90 days** from acknowledgement.
- Credit (or anonymity, if you prefer) once a fix ships.

For broader context on Configurite's security architecture, threat model,
and operational guidance, read [`docs/en/security.md`](docs/en/security.md)
([Türkçe](docs/tr/guvenlik.md)).
