## 🎉 Configurite v0.1 — First Public Release

> Drop-in replacement for `appsettings.json`. AES-256-GCM encryption at rest, hot reload, audit log, web admin UI, and a `dotnet` global tool — four NuGet packages shipping in lockstep.

### 🇬🇧 English

This is the first public release of Configurite. If you've ever seen `appsettings.Production.json` carrying real database passwords in plaintext, this is the package built for you.

#### What's in the box

| Package | Purpose | Size |
|---|---|---|
| **Configurite** | Configuration provider library | ~30 KB |
| **Configurite.Cli** | `dotnet configurite` global tool — migrate, rotate, audit, export | ~13 MB |
| **Configurite.AdminUI** | Drop-in web admin panel (bilingual EN/TR) | ~20 KB |
| **Configurite.Postgres** | `IConfiguriteStore` implementation backed by PostgreSQL | ~10 KB |

All four ship in **lockstep** — installing `Configurite 8.x` means `Configurite.AdminUI 8.x` is source-compatible.

#### Highlights

- 🔐 **AES-256-GCM** encryption per value (PBKDF2-HMAC-SHA256, 200 K iterations)
- 🔄 **Hot reload** via `FileSystemWatcher` and `IChangeToken`
- 🌍 **Environment-aware** overrides (Development / Production / …) in a single database
- 📦 **One-shot migration** from existing `appsettings*.json` files
- 🔁 **Atomic master-key rotation** in a single SQLite transaction
- 🧰 **Dotnet tool**: 9 commands (init / migrate / rotate / get / set / list / delete / audit / export)
- 🖼️ **Web admin UI** with audit log, mounted on any ASP.NET Core endpoint
- 📊 **Observability**: ILogger + OpenTelemetry traces & metrics out of the box

#### 30-second quick start

```csharp
using Configurite;

var builder = WebApplication.CreateBuilder(args);

builder.Configuration.AddConfigurite(opts =>
{
    opts.DatabasePath   = "appsettings.db";
    opts.Environment    = builder.Environment.EnvironmentName;
    opts.ReloadOnChange = true;
});

var app = builder.Build();
app.Run();
```

That's it. `IConfiguration["ConnectionStrings:Default"]` returns a decrypted string transparently. `IOptionsMonitor<T>` reacts to file changes through the standard hot-reload pipeline.

#### Migrating from `appsettings.json`

```bash
dotnet configurite migrate ./appsettings.db ./config-folder \
    --encrypt "ConnectionStrings:*" \
    --encrypt "*:Password" \
    --encrypt "*:ApiKey"
```

One command. Reads every `appsettings*.json` in the folder, flattens to `:`-separated keys, infers the environment from each filename, encrypts paths matching the glob patterns, and writes everything to `appsettings.db`.
